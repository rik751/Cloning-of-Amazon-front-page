# Cloning-of-Amazon-front-page
Project Overview
The goal of this project was to practice front-end development by reconstructing a complex, high-traffic user interface. It focuses on implementing a responsive navigation bar, category sections, and a structured layout.

# Key Features
Custom Navigation Bar: Features a logo section, location selector, multi-category search dropdown, and user account links.

Interactive Elements: Includes hover effects on navigation items and search components that change border colors for better user feedback.

Product Grid Layout: Uses Flexbox to organize various shopping categories such as Gaming Accessories, Home Essentials, and Fashion Deals.

Hero Section: Displays a prominent gaming-themed banner with a call-to-action for regional store switching.

Footer: A clean, dark-themed footer that includes creator credits and regional identification.

# Technologies Used
HTML5: Used for semantic structure of the homepage components.

CSS3: Implemented for layout (Flexbox), animations (@keyframes), and interactive styling.

Font Awesome: Integrated for scalable vector icons like the location pin and hamburger menu.

# How to Run
Clone this repository to your local machine.

Ensure all image assets (amazon.png, game_amazon.jpg, language.png) are in the same directory as the HTML file.

Open amazon index.html in any modern web browser.
